# bg2 vr viewer

Currently only works on  Unreal Engine v4.24.

1- Checkout the appropiate version of the bg2-unreal-tools and bg2-vr-tools inside the Plugins directory

```other
bg2-vr-viewer
|- Plugins
|-- Bg2UnrealTools   > checkout the 4.24 branch
|-- Bg2VRTools       > checkout the ue4.24 branch
|...
```

2- Generate visual studio 2019 project files and build
